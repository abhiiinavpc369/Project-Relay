## ui-ux.md (UI / UX Guidelines)

### Design Theme

* Dark‑first UI | Can be changed to light mode in settings
* Purple / Blue accent
* Soft animations

### UX Rules

* One‑hand usage
* No unnecessary popups
* Clear read receipts

### Screens

* Login / Signup
* Chat list
* Chat screen
* Settings
* Status screen (Whatsapp like)
